Practica desafios de codigo tipo entrevista con ejercicios progresivos y autoevaluacion aislada en JavaScript o Python.

## Instalacion en un clic (recomendado)

Elige la version que quieras resolver y abre ese repositorio en Codespaces:

```bash
JavaScript: https://github.com/breatheco-de/interview-coding-challenge-javascript
Python: https://github.com/breatheco-de/interview-coding-challenge-python
```

## Instalacion local

1. Clona o abre el repositorio del lenguaje con el que quieras trabajar:
```bash
git clone https://github.com/breatheco-de/interview-coding-challenge-javascript.git
# o
git clone https://github.com/breatheco-de/interview-coding-challenge-python.git
```
2. Sigue las instrucciones de instalacion del README de ese repositorio:
```bash
JavaScript README: https://github.com/breatheco-de/interview-coding-challenge-javascript
Python README: https://github.com/breatheco-de/interview-coding-challenge-python
```

## Estructura de ejercicios

El challenge esta disponible en dos repositorios separados:

1. `interview-coding-challenge-javascript`: Resuelve los ejercicios en JavaScript.
2. `interview-coding-challenge-python`: Resuelve los ejercicios en Python.

Ambos repositorios incluyen:

1. `exercises/00-welcome`: Paso introductorio solo con readme.
2. `exercises/01-...` hasta `exercises/15-...`: Retos de codigo.

Dependiendo del lenguaje elegido, la mayoria de carpetas de ejercicios incluye:

1. Version JavaScript: `app.js`, `tests.js` y algunos `solution.hide.js`.
2. Version Python: `app.py`, `test.py` y algunos `solution.hide.py`.
3. `README.md`: Instrucciones en ingles.
4. `README.es.md`: Instrucciones en espanol.

## Notas

- Este repositorio es un wrapper que apunta a las versiones JavaScript y Python.
- Mantener `README.md` y `README.es.md` sincronizados cuando se cambie contenido.
- Los archivos de estudiantes estan intencionalmente sin resolver en los repositorios por lenguaje.
