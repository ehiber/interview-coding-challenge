# Interview Coding Challenges

[View in English](./README.md)

Practica desafios de codigo tipo entrevista con ejercicios progresivos.

## Antes de Empezar

1. Instala LearnPack globalmente:
`npm install @learnpack/learnpack -g`
2. Inicia los ejercicios desde la raiz del proyecto:
`learnpack start`

## Estructura del Proyecto

- `exercises/01-hello-world` hasta `exercises/15-maximum-subarray-sum`
- Cada ejercicio incluye:
- `README.md` (instrucciones en ingles)
- `README.es.md` (instrucciones en espanol)
- `app.js` (starter para estudiantes)
- `tests.js` (validaciones automaticas)
