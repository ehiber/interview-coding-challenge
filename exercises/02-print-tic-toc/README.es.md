# `02` TicToc

Escribe un programa que imprima los numeros del 1 al 100 y:

- Para los multiplos de `3`, imprima `"Tic"` en lugar del numero.
- Para los multiplos de `5`, imprima `"Toc"`.
- Si el numero es multiplo de 3 y 5, imprima `"TicToc"`.

Ejemplo de salida:

```sh
1
2
Tic
4
Toc
...
14
TicToc
```
