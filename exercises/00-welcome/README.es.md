# `00` Elige Tu Lenguaje

Bienvenido a **Interview Coding Challenges**.

Este repositorio contiene dos variantes separadas de LearnPack:

1. `javascript/`
2. `python/`

## Que hacer ahora

Primero elige un lenguaje y luego abre una terminal para ejecutar LearnPack desde esa carpeta:

```bash
cd javascript
learnpack start
```

O:

```bash
cd python
learnpack start
```

## Importante

- Este tutorial del root solo funciona como selector de lenguaje para Codespaces y LearnPack.
- Los ejercicios reales estan dentro de `javascript/` y `python/`.
- Usa solo una ruta de lenguaje a la vez.
